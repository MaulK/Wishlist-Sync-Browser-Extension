# Icons Directory

This directory contains the extension icons. You need to create the following PNG images:

## Required Icons

1. **icon16.png** (16x16 pixels)
   - Used for browser toolbar
   - Simple version of the main icon

2. **icon48.png** (48x48 pixels)
   - Used for extension management page
   - Medium detail version

3. **icon128.png** (128x128 pixels)
   - Used for Chrome Web Store
   - High detail version

## Icon Design Guidelines

- **Color Scheme**: Use gradient from #667eea (purple-blue) to #764ba2 (purple)
- **Style**: Modern, clean, and recognizable
- **Symbol**: Consider using a heart, wishlist, or shopping bag symbol
- **Background**: Transparent PNG format
- **Format**: 24-bit PNG with alpha channel

## Icon Design Suggestion

A simple heart icon with a shopping bag or wishlist symbol inside, using the brand colors:

```
Background: Gradient from #667eea to #764ba2
Symbol: White heart with a small shopping bag
Border: Subtle white border for contrast
```

## Creating the Icons

You can use these tools to create the icons:

1. **Figma** (free design tool)
2. **Canva** (online design tool)
3. **GIMP** (free image editor)
4. **Adobe Illustrator** (paid)
5. **Online icon generators**

## Temporary Placeholder

For testing purposes, you can use any square PNG images. The extension will work with any images as long as they have the correct filenames and dimensions.

## After Creating Icons

1. Replace the placeholder images with your new icons
2. Reload the extension in developer mode
3. Verify icons appear correctly in:
   - Browser toolbar
   - Extensions page
   - Popup and dashboard headers
