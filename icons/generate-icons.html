Wishlist Sync Icon

Right-click on the icon below and save as PNG for each size

How to Create PNG Icons:

  1. Right-click on the icon above
  2. Select "Save Image As..."
  3. Save as icon16.png (resize to 16x16)
  4. Save as icon48.png (resize to 48x48)
  5. Save as icon128.png (resize to 128x128)

Alternative Methods:

  1. Open icon.svg in a design tool (Figma, Canva, etc.)
  2. Export as PNG at the required sizes
  3. Use online SVG to PNG converters
  4. Use browser DevTools to screenshot the SVG
